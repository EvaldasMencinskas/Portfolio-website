Unity game

This game is about fighting enemies to gain experience points and get stronger and try to go as far as possible in an endless side scroller type game

Gameplay

the game revolves around improving the character's stats and leveling up by walking in two-dimensional space, fighting enemies using close combat techniques, and minimal obstacles overcoming attempts to go as far as possible.

Mechanics

Rpg game genre system with levelling, improving statistics and gaining experience from defeated enemies. Minimum obstacle courses. as the character moves in the upper right corner, the number of meters will increase, indicating how far the character has travelled, based on that number, bosses will appear and the environment will change. Optional equipment system with inventory, equipment delivery to get a statistics bonus but without the ability to change the look, collect items from defeated enemies and a mini home area with storage space.
